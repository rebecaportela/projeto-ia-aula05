# Verificador de Números Primos - Tópicos ## 📋 Estrutura Geral - **Função**: `eh_primo(numero)` - verifica se um número é primo - **Entrada**: solicita um número ao usuário - **Saída**: mensagem indicando se o número é primo ou não ## 🔧 Função `eh_primo(numero)` ### Validação Inicial - Verifica se `numero <= 1` - Números menores ou iguais a 1 **não são primos** - Retorna `False` imediatamente ### Loop de Verificação - Itera de **2 até a raiz quadrada do número** - Usa `range(2, int(numero ** 0.5) + 1)` - Otimização: reduz iterações testando apenas até √número ### Teste de Divisibilidade - Usa operador módulo: `numero % i == 0` - Se encontrar um divisor, retorna `False` - Número não é primo ### Resultado Final - Se nenhum divisor for encontrado, retorna `True` - Número **é primo** ## 💻 Execução Principal ### Entrada do Usuário - `int(input("Digite um número: "))` - Converte input (string) para inteiro ### Condicional Final - Chama `eh_primo(numero)` - **Se True**: exibe "É primo" - **Se False**: exibe "Não é primo" ## ⚡ Eficiência do Algoritmo - Não testa todos os números até o próprio número - Testa apenas até √número - Exemplo: para 100, testa apenas até 10 - Muito mais rápido para números grandes
